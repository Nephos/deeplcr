# deeplcr

Deepl in command line with crystal

## Installation

`make`

## Usage

    ./deeplcr -f EN -t FR -a "I love testing new things"
    FROM: EN
    TO  : FR
    TEXT: I love testing new things
    -------------------------------
    J'adore tester de nouvelles choses
